# 의존 라이브러리 버전 올리기

1. `build.gradle`에 선언된 현재 버전을 확인한다.
2. 각 라이브러리의 최신 안정 버전과 릴리스 노트를 조사한다.
3. 한 번에 한 라이브러리씩 버전을 올리고 `./gradlew test`로 검증한다.
4. 테스트가 실패하면 릴리스 노트에서 호환성 변경을 찾아 코드를 고친다.
5. 올린 버전과 주요 변경 사항을 정리해서 보고한다.
